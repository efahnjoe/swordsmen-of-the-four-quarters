**人物设定表**

> **说明**：本表适用于主角、重要配角、敌对势力、宗门高层等核心角色。重点突出“反派出身”“修仙体系”“命运转折”三大特色。

| 类别           | 字段              | 字段内容 | 说明                                                           |
| -------------- | ----------------- | -------- | -------------------------------------------------------------- |
| **基础信息**   | 姓名              |          | 真名（如“宁志”），可附化名/道号（如“血瞳子”）                  |
|                | 外号 / 恶名       |          | 他人对其的称呼，体现其“反派”标签（如“屠城魔君”“噬心老祖”）     |
|                | 性别              |          | 男 / 女 / 非二元（修仙界可有特殊设定）                         |
|                | 出场年龄          |          | 实际年龄 + 外貌年龄（修仙者常驻颜）<br>例：327岁（外表约25岁） |
|                | 身高 / 体型       |          | 可加入灵压、气场描述（如“身形瘦削，周身隐有黑焰缭绕”）         |
|                | 相貌特征          |          | 发色、瞳色、疤痕、异象（如“左眼封印着上古凶兽之瞳”）           |
| **修仙根基**   | 体质 / 血脉       |          | 特殊体质（如“九幽冥骨体”“混沌魔胎”）或神兽血脉                 |
|                | 灵根属性          |          | 金木水火土 + 变异灵根（如“纯阴煞灵根”“伪灵根（后觉醒）”）      |
|                | 修为境界          |          | 当前境界（如“元婴后期”），可注明瓶颈或隐患                     |
|                | 所属势力          |          | 原宗门 / 现归属 / 自立山头（如“原青云宗外门→自创‘幽冥教’”）    |
|                | 身份地位          |          | 在势力中的角色（如“叛徒”“幕后黑手”“被迫继任的魔宗少主”）       |
| **功法与手段** | 主修功法          |          | 名称 + 特性（如《万劫噬心诀》——以情入道，吞噬执念）            |
|                | 标志神通 / 秘术   |          | 战斗或生存依赖的核心能力（如“血影遁”“言出法随（残缺）”）       |
|                | 本命法宝 / 兵器   |          | 名称 + 来历 + 灵性（如“断罪剑——由仇人脊骨炼成，嗜主”）         |
|                | 弱点 / 功法反噬   |          | 修习代价（如“每用一次魔功，记忆消散一分”）                     |
| **过往与心魔** | 出身背景          |          | 凡人世家 / 修仙弃子 / 转世重修 / 实验产物等                    |
|                | 关键转折事件      |          | 导致其“成为反派”的导火索（如“被师门献祭给上古邪神”）           |
|                | 最大执念          |          | 驱动其行动的核心（如“证明自己不是灾星”“复活亡妻”）             |
|                | 心魔具象          |          | 内心阴影的象征（如“总梦见幼时村庄焚毁的火光”）                 |
|                | 是否重生 / 穿越   |          | 若是，说明前世身份与今生目标差异                               |
| **性格与行为** | 表面性格          |          | 对外展现的形象（如“温文尔雅的宗门长老”）                       |
|                | 真实性格          |          | 内在本质（如“冷酷算计，但对小动物异常温柔”）                   |
|                | 道德底线          |          | 能做什么 / 绝不做什么（如“可杀人夺宝，但不伤孩童”）            |
|                | 习惯动作 / 口头禅 |          | 增强辨识度（如“说话时指尖无意识划出血痕”）                     |
|                | 对“正道”的态度    |          | 蔑视 / 利用 / 渴望回归 / 彻底对立                              |
| **人际关系**   | 重要羁绊者        |          | 师尊、挚友、仇敌、红颜等（注明关系现状）                       |
|                | 敌对势力          |          | 主要对手及其动机（如“天机阁欲除之而后快，因其知晓秘辛”）       |
|                | 是否有追随者      |          | 追随者类型（死士 / 被蛊惑者 / 利益同盟）                       |
|                | 在主角团中的定位  |          | 若为主角，则填“反派主角”；若为配角，填“亦敌亦友”等             |
| **命运与成长** | 初始立场          |          | 故事开篇时的身份（如“人人喊打的大魔头”）                       |
|                | 核心目标          |          | 短期 + 长期（如“夺取九幽地脉→超脱轮回”）                       |
|                | 可能的救赎路径    |          | 是否存在洗白/转变可能？如何触发？                              |
|                | 与天道 / 气运关系 |          | 被天道厌弃？身负大气运？逆天改命者？                           |
|                | 结局倾向          |          | 悲剧 / 顿悟 / 成就新道 / 与世界同归于尽                        |
